# EdStats Dashboard

Interactive Streamlit dashboard built using the World Bank Global Education Statistics dataset.

## Features

* Pie Chart
* Histogram
* Line Chart
* Bar Chart
* Scatter Plot
* Box Plot
* Heatmap
* Area Chart
* Count Plot
* Violin Plot

## Technologies

* Python
* Streamlit
* Pandas
* Matplotlib
* Seaborn

## Project Structure

EdStats-Dashboard/

├── app.py

├── charts.py

├── filters.py

├── requirements.txt

├── README.md

└── data/

  └── EdStatsData.csv

## Run Locally

pip install -r requirements.txt

streamlit run app.py
